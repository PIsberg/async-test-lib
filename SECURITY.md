# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

To report a vulnerability, open a [GitHub Security Advisory](https://github.com/PIsberg/async-test-lib/security/advisories/new) in this repository. This keeps the disclosure private until a fix is available.

Include as much of the following as possible:

- Type of issue (e.g. dependency confusion, code injection, path traversal)
- Steps to reproduce
- Affected versions
- Any suggested remediation

You can expect an acknowledgement within **7 days** and a resolution timeline within **90 days** depending on severity.

## Preferred Languages

Reports in English or Swedish are both welcome.
